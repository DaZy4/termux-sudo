# termux-sudo

**Requirements:**

-> Root access on device (Magisk & installed patched boot.img)

**Installation:**
```
wget https://raw.githubusercontent.com/DaZy4/termux-sudo/main/install.sh && chmod +x install.sh && ./install.sh && rm install.sh
```

**Usage:**
```
sudo su
```
